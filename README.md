# YouTube live low-latency

## Description

Adjusts YouTube playback speed when you're behind on a live stream, attempting to keep you at low live latency in a better way than YouTube's default behavior.

## Install

Install a userscript manager like [Violentmonkey](https://violentmonkey.github.io/), then click to install: [YouTube live low-latency](https://raw.githubusercontent.com/SilentShout42/youtube-live-low-latency/refs/heads/main/youtube-live-low-latency.user.js)
